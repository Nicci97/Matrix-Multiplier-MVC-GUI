1. Download JavaFX library from https://gluonhq.com/products/javafx/
2. Add the JavaFX lib folder as a library in your IDE
3. Add a VM option to the run configurations, use the following: --module-path <absolute-path-to-above-mentioned-lib-folder> --add-modules javafx.controls,javafx.fxml

The project setup on github contains files for IntelliJ IDE
